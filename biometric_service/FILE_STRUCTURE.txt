QUICK REFERENCE
===============

Start the Service:
    cd /workspace/biometric_service
    ./start.sh

Check Dependencies:
    python check_dependencies.py

Run Tests:
    python test_api.py

API Documentation:
    http://localhost:8000/docs
    http://localhost:8000/redoc

Try Frontend Demo:
    Open examples/frontend_integration.html in browser


KEY ENDPOINTS
=============

POST   /face/register              - Register a face
POST   /face/authenticate          - Authenticate by face
POST   /face/quality-check         - Check image quality
GET    /face/check-registration/{user_id}  - Check registration
DELETE /face/reset/{user_id}       - Reset registration
POST   /fingerprint/register       - Register fingerprint
POST   /fingerprint/authenticate   - Authenticate fingerprint
GET    /fingerprint/simulate       - Simulate scan
GET    /health                     - Health check
GET    /                           - Service info


DEPLOYMENT OPTIONS
==================

1. Local Development:
   ./start.sh

2. Manual Setup:
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   uvicorn app:app --reload

3. Docker:
   docker-compose up

4. Production (systemd + nginx):
   See DEPLOYMENT.md
